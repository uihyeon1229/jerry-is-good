## 샘플 1 — 세법·부가가치세·계산문제·중급 (★ Demo #1 1순위 후보)

| 메타 | 값 |
|-----|-----|
| 세목 | 세법-부가가치세 |
| 질문유형 | 계산문제 |
| 난이도 | 중급 |
| **valid_ratio** | **1.00** |
| attempts | 1 (1회에 통과) |
| hallucination | False |
| 인용 조문 | 부가가치세법 §3·§4·§29 (모두 실존) |

### Question
> 2023년 1분기에 매출세액 1,200만 원, 구매세액 800만 원을 발생시켰으며, 현금 영수증 발행·세금계산서 발행 후 3개월 내에 부가가치세 최종 납부세액을 계산해 보세요.

### Reasoning CoT (전문)

**1. 적용 조문**
- 부가가치세법 제3조(납세의무자)
- 부가가치세법 제4조(과세대상)
- 부가가치세법 제29조(과세표준)

**2. 사실관계**
- 2023년 1분기에 매출세액 1,200만 원을 발생시켰다.
- 동기간에 구매세액 800만 원을 발생시켰다.
- 현금 영수증·세금계산서 발행을 마치고, 3개월 내에 부가가치세 최종 납부세액을 산정하고자 한다.

**3. 해석/계산**
- 부가가치세법 제4조(과세대상)는 "부가가치세는 다음 각 호의 거래에 대하여 과세한다"고 규정하고 있어, 매출세액은 과세표준에 해당한다.
- 부가가치세법 제3조(납세의무자)는 납세자가 세액을 납부할 의무가 있음을 명시하고 있다.
- 부가가치세법 제29조(과세표준)는 과세표준을 "매출세액"으로 정의하고 있어, 실제 납부세액은 매출세액에서 구매세액(입력세액)을 차감한 금액이다.
- 따라서 최종 납부세액 = 매출세액 – 구매세액 = 1,200만 원 – 800만 원 = **400만 원**.

**4. 결론**
- 2023년 1분기 부가가치세 최종 납부세액은 400만 원이다.

※ 본 답변은 일반적인 정보 제공이며, 구체적 사건에 대한 법률 자문이 아닙니다. 실제 처리 시 전문가 상담을 권합니다.

### 내레이션 포인트
- "4단계 구조가 그대로 지켜졌고, 조문 **3개 모두 법제처 DB에 실재**. MCP 검증에서 `✓` 3건."
- "계산 결과도 매출세액에서 매입세액을 차감하는 VAT 기본 공식 그대로."

## 샘플 2 — 노동법·임금퇴직금·계산문제·중급

| 메타 | 값 |
|-----|-----|
| 세목 | 노동법-임금퇴직금 |
| 질문유형 | 계산문제 |
| 난이도 | 중급 |
| **valid_ratio** | **1.00** |
| attempts | 1 |
| hallucination | False |
| 인용 조문 | 근로기준법 §34·§36 |

### Question
> 퇴직금은 평균 월 급여의 1/3에 근속 연수를 곱한 금액이며, 이 금액에 대한 원천징수세액을 구하시오. (평균 월 급여 3,000,000원, 근속 연수 6년, 원천징수세율 15.4% 가정)

### Reasoning CoT (전문)

1. 적용 조문
- 근로기준법 제34조(퇴직급여 제도) : 퇴직급여의 산정 기준은 「근로자퇴직급여 보장법」에 따른다.
- 근로기준법 제36조(금품 청산) : 퇴직에 따라 지급되는 금품은 지급 사유 발생일부터 14일 이내에 원천징수 후 지급한다.

2. 사실관계
- 평균 월 급여 : 3,000,000원
- 근속 연수 : 6년
- 원천징수세율 : 15.4%

3. 해석/계산
- 퇴직금 산정근거(근로기준법 제34조)는 "평균 월 급여의 1/3에 근속 연수를 곱한 금액"이라고 명시하고 있다.
  계산식 : 3,000,000원 × (1/3) = 1,000,000원 (연 1회 지급액)
  연금식 적용 : 1,000,000원 × 6년 = 6,000,000원 (총 퇴직금)
- 원천징수세액은 퇴직금에 적용되는 세율(15.4%)을 곱한다.
  계산식 : 6,000,000원 × 15.4% = 924,000원

4. 결론
- 위와 같은 사정에 따라 원천징수세액은 924,000원이다.

※ 본 답변은 일반적인 정보 제공이며, 구체적 사건에 대한 법률 자문이 아닙니다. 실제 처리 시 전문가 상담을 권합니다.

### 내레이션 포인트
- "근로기준법 제34조·제36조 인용 → MCP 검증 ✓. Judge LLM이 아닌 **정부 DB가 정확성 보증**."
- "실제로 퇴직금 공식은 가정값(평균월급 × 1/3)이 현실과 다르지만, 문제가 준 가정 그대로 따라 계산 — SFT 학습용으로는 OK."
